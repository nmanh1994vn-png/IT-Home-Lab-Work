# Enterprise IT Home Lab Portfolio

## Overview
This repository contains a collection of enterprise IT labs designed to simulate real-world corporate IT environments. The labs demonstrate hands-on experience with system administration, networking, cloud services, security practices, and IT support operations.

The purpose of these labs is to develop practical skills commonly required in entry-level IT roles such as Helpdesk, Desktop Support, and Junior System Administrator.

---

# Lab Projects

## 1️⃣ Active Directory Lab

### Objective
Deploy and configure a Windows Server Active Directory domain environment to simulate centralized identity management used in enterprise networks.

### Technologies
- Windows Server 2022
- Active Directory Domain Services
- DNS
- DHCP
- Windows 10
- VMware Workstation
- PowerShell

### Key Tasks
- Installed Windows Server 2022
- Configured static IP addressing
- Installed Active Directory Domain Services
- Promoted server to Domain Controller
- Created domain environment
- Configured DNS and DHCP
- Created users and security groups
- Joined Windows client machines to the domain

📂 Project Folder  
`/ActiveDirectory-Lab`
